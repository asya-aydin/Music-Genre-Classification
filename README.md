# Music-Genre-Classification Decision Tree vs. k-NN

This project compares the performance of **Decision Tree** and **k-Nearest Neighbors (k-NN)** algorithms for classifying music genres using the GTZAN dataset.

## Dataset
The project uses the **GTZAN Dataset - Music Genre Classification** from Kaggle.
- **Source:** [Kaggle GTZAN Dataset](https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification)
- **File Used:** `features_3_sec.csv` (contains 9,990 samples with 58 audio features).

- ## Installation
To set up the environment, ensure you have Python 3.x installed, then run:

```bash
pip install -r requirements.txt
```

- ## How to run
To train the models and generate evaluation metrics and charts, simply execute:

```bash
python main.py
